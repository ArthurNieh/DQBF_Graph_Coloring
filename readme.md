# DQBF Graph coloring

This project proposed to solve graph coloring problem using state-of-the-art DQBF solver.

Author: ArthurNieh, cycloud0203, Hayooota

#### Build the solver 

need to beunder Linux environment

```shell
cd pedant-solver
mkdir build && cd build
cmake ..
make
```

## generate blif file

```shell
python3 blif_gen.py
```
